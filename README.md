## What is raiders-go? 
raiders(https://github.com/Sushimania/raiders)라는 간단한 프로그램이 확장된 프로젝트가 raiders-go 입니다. 블록체인 경제를 활용하여 컴퓨터 해시파워를 동원해 비트코인 개인키를 무작위로 생성하여 비트코인을 찾는 프로그램입니다.
